# AI 食譜網站（Production Ready）

## 專案啟動

1. 安裝相依套件
   ```sh
   npm install
   ```
2. 開發模式啟動
   ```sh
   npm run dev
   ```
3. 打包 production 靜態檔案
   ```sh
   npm run build
   ```
4. 預覽 production build
   ```sh
   npm run preview
   ```

## 目錄結構

- `public/index.html`：入口 HTML
- `src/components/RecipeCard.jsx`：食譜卡元件
- `src/App.jsx`：主 App 元件
- `src/main.jsx`：React 入口
- `src/index.css`：Tailwind CSS 設定

## 技術棧
- React 17
- Vite
- Tailwind CSS
